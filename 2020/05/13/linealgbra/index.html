线性代数学习总结*
Created 2020-05-13| Updated 2020-05-28|学习总结
Post View:|Comments:

本文为大一下学期学习线性代数后的总结博客。本文并不注重罗列所有的知识点，而是希望通过某种方式将目前所学的线性代数内容串联起来。本文将包括但不仅限于课本内容。文章在遇到关键或是有趣的地方会进行稍加详细的说明。写文章的主要目的是方便以后的复习。欢迎在评论区与本人交流。鉴于作者水平有限，如若内容有误，还请读者及时批评指正。

初探线性空间

本文的展开，将从线性空间开始。

默认本文的读者已经对线性代数有所了解。

向量空间

我们知道，平面中的点对应一个二元有序实数对，空间中的点对应一个三元有序实数对。根据这些，我们把通常空间的概念加以推广。

设 n ~n~ n 是正整数，由 n ~n~ n 个实数 a1, a2, …, an ~a_1,~a_2,~\ldots,~a_n~ a1​, a2​, …, an​ 组成的有序 n ~n~ n 元数组

α=(a1, a2, …, an)\alpha = (a_1,~a_2,~\ldots,~a_n) α=(a1​, a2​, …, an​)

称为 n ~n~ n 维实向量。所有 n ~n~ n 维实向量的集合记为 Rn ~\R^n~ Rn .

下面定义向量的加法，使它满足一些公理。定义

α+β=(a1+b1, a2+b2, …, an+bn)\alpha + \beta = (a_1 + b_1,~a_2 + b_2,~\ldots,~a_n + b_n) α+β=(a1​+b1​, a2​+b2​, …, an​+bn​)

显然 α+β∈Rn ~\alpha + \beta \in \R^n~ α+β∈Rn , 这样便规定了 Rn ~\R^n~ Rn 中向量的加法运算。

 Rn ~\R^n~ Rn 中向量的加法满足下面的公理

  • 加法结合律

    (α+β)+γ=α+(β+γ)(\alpha + \beta) + \gamma = \alpha + (\beta + \gamma) (α+β)+γ=α+(β+γ)

  • 加法交换律

    α+β=β+α\alpha + \beta = \beta + \alpha α+β=β+α

  • 存在零向量

    α+0=α\alpha + \boldsymbol{0} = \alpha α+0=α

  • 存在负向量

    α+β=0\alpha + \beta = \boldsymbol{0} α+β=0

纯量与 Rn ~\R^n~ Rn 中向量的乘法规定如下：设 λ∈R, α=(a1, a2, …, an)∈Rn ~\lambda\in\R,~\alpha = (a_1,~a_2,~\ldots,~a_n)\in\R^n~ λ∈R, α=(a1​, a2​, …, an​)∈Rn , 则

λα=(λa1, λa2, …, λan)\lambda\alpha = (\lambda a_1,~\lambda a_2,~\ldots,~\lambda a_n) λα=(λa1​, λa2​, …, λan​)

纯量与向量的乘法满足以下公理

  • 结合律

    (λμ)α=λ(μα)(\lambda\mu)\alpha = \lambda(\mu\alpha) (λμ)α=λ(μα)

  • 存在乘法单位元

    1⋅α=α1\cdot\alpha = \alpha 1⋅α=α

向量的加法和纯量与向量的乘法之间有以下关系

  • 乘法对向量加法的分配律

    λ(α+β)=λα+λβ\lambda(\alpha + \beta) = \lambda\alpha + \lambda\beta λ(α+β)=λα+λβ

  • 乘法对纯量加法的分配律

    (λ+μ)α=λα+μα(\lambda + \mu)\alpha = \lambda\alpha + \mu\alpha (λ+μ)α=λα+μα

以上八条公理是作为向量空间需要满足的基本条件，向量加法和标量乘法也可以认为是据此定义的。

 Rn ~\R^n~ Rn 连同向量的加法以及纯量与向量的乘法一起被称为 n ~n~ n 维实向量空间。对于任意数域 F ~\mathbb{F}~ F 上的 n ~n~ n 维向量空间，其定义与 Rn ~\R^n~ Rn 相同。

线性空间

上面我们定义了向量空间，并且定义了向量集合上的加法和数乘。下面我们把集合的两种运算连同它们满足的公理加以概括抽象，便引出线性空间的概念。

设 V ~V~ V 是一个非空集合，它的元素为向量。设 F ~\mathbb{F}~ F 是一个数域，它的元素称为纯量。在 V ~V~ V 中定义了向量的加法，即对任意 α, β∈V ~\alpha,~\beta\in V~ α, β∈V ,  V ~V~ V 中有唯一的向量与 α+β ~\alpha + \beta~ α+β 对应，向量 α+β ~\alpha + \beta~ α+β 称为向量 α ~\alpha~ α 与 β ~\beta~ β 的和；在 V ~V~ V 中还定义了纯量与向量的乘法，即对任意纯量 λ∈F ~\lambda\in\mathbb{F}~ λ∈F , 向量 α∈V ~\alpha\in V~ α∈V ,  V ~V~ V 中有唯一的向量 λV ~\lambda V~ λV 与之对应，向量 λα ~\lambda\alpha~ λα 称为纯量 λ ~\lambda~ λ 与向量 α ~\alpha~ α 的积。如此，则集合 V ~V~ V 称为数域 F ~\mathbb{F}~ F 上的线性空间。

看似我们只要求了向量的加法和数乘得到的结果仍在 V ~V~ V 中，但由于我们先前已经定义过了向量的加法和数乘，所以它们自然的应当满足那八条公理。

同时要注意到，定义向量空间时，我们要求其中的元素为有序 n ~n~ n 元数组，但在此时定义的线性空间，我们并没有对集合 V ~V~ V 的内容做要求。也就是说 V ~V~ V 中的元素可以是任何东西，只需满足上面定义的加法和标量乘法即可。

如此，线性空间的概念是很广阔的。比如，函数可以构成线性空间，矩阵可以构成线性空间，诸如此类。

线性相关

设 V ~V~ V 是数域 F ~\mathbb{F}~ F 上的线性空间， S⊆V ~S\subseteq V~ S⊆V . 如果存在向量 α1, α2, …, αk∈S ~\alpha_1,~\alpha_2,~\ldots,~\alpha_k\in S~ α1​, α2​, …, αk​∈S , 和不全为零的纯量 λ1, λ2, …, λk∈F ~\lambda_1,~\lambda_2,~\ldots,~\lambda_k\in\mathbb{F}~ λ1​, λ2​, …, λk​∈F 使得

λ1α1+λ2α2+⋯+λkαk=0\lambda_1\alpha_1 + \lambda_2\alpha_2 + \cdots + \lambda_k\alpha_k = \boldsymbol{0} λ1​α1​+λ2​α2​+⋯+λk​αk​=0

则向量集合 S ~S~ S 称为线性相关的。特别的，当向量集合 S ~S~ S 由有限个向量 α1, α2, …, αk ~\alpha_1,~\alpha_2,~\ldots,~\alpha_k~ α1​, α2​, …, αk​ 组成时，也称向量 α1, α2, …, αk ~\alpha_1,~\alpha_2,~\ldots,~\alpha_k~ α1​, α2​, …, αk​ 线性相关。不是线性相关的向量集合 S ~S~ S 称为线性无关的。

此定义常常用来证明线性相关或是线性无关。

设 S, T⊆V ~S,~T\subseteq V~ S, T⊆V . 如果向量集合 T ~T~ T 中的每个向量都可以由向量集合 S ~S~ S 线性表出，则称向量集合 T ~T~ T 可由向量集合 S ~S~ S 线性表出。若两个向量集合可以相互线性表出，则它们是等价的。等价关系满足三条性质，即自反性，对称性和传递性。

若两个向量集合等价，那么它们的维数相同。

设向量集合 S={α1, α2, …, αs}⊆V ~S = \{\alpha_1,~\alpha_2,~\ldots,~\alpha_s\}\subseteq V~ S={α1​, α2​, …, αs​}⊆V . 如果 S ~S~ S 中的向量 β1, β2, …, βt ~\beta_1,~\beta_2,~\ldots,~\beta_t~ β1​, β2​, …, βt​ 线性无关，并且对任意的 β∈S ~\beta\in S~ β∈S , 向量 β, β1, β2, …, βt ~\beta,~\beta_1,~\beta_2,~\ldots,~\beta_t~ β, β1​, β2​, …, βt​ 线性相关，则称向量 β1, β2, …, βt ~\beta_1,~\beta_2,~\ldots,~\beta_t~ β1​, β2​, …, βt​ 是向量 α1, α2, …, αs ~\alpha_1,~\alpha_2,~\ldots,~\alpha_s~ α1​, α2​, …, αs​ 的一个极大线性无关组。极大线性无关组中的向量个数 r ~r~ r 称为向量 α1, α2, …, αs ~\alpha_1,~\alpha_2,~\ldots,~\alpha_s~ α1​, α2​, …, αs​ 的秩

极大线性无关组是个略带矛盾的概念。它既要求线性无关，又要求向量的数量尽可能的多。

基与坐标

在二维，三维的情况下，我们已经熟知坐标系和基。现在我们将这些概念推广到线性空间中。

设 S ~S~ S 是数域 F ~\mathbb{F}~ F 上的线性空间 V ~V~ V 的向量集合。如果向量集合 S ~S~ S 线性无关，而且 V ~V~ V 中的每个向量都可以由向量集合 S ~S~ S 线性表出，则向量集合 S ~S~ S 称为线性空间 V ~V~ V 的一组基， S ~S~ S 中的向量称为基向量．

同构

对于一个数域 F ~\mathbb{F}~ F , 其上的有限维线性空间当然有很多。因此，自然希望能对数域 F ~\mathbb{F}~ F 上的所有有限维线性空间的集合进行分类，使得同属于一个类的线性空间具有相同的结构。

设 V1 ~V_1~ V1​ 与 V2 ~V2~ V2 是同一个数域 F ~\mathbb{F}~ F 上的两个线性空间．如果存在 V1 ~V1~ V1 到 V2 ~V2~ V2 上的一一对应 η ~\eta~ η ，它把 V1 ~V1~ V1 中的向量 α ~\alpha~ α 映为 V2 ~V2~ V2 的向量 η(α) ~\eta(\alpha)~ η(α) ，使得对任意 α, β∈V1, λ∈F,  ~\alpha,~\beta\in V_1,~\lambda\in\mathbb{F},~~ α, β∈V1​, λ∈F,  都有

η(α+β)=η(α)+η(β)η(λα)=λη(α)\begin{aligned} \eta(\alpha + \beta) &= \eta(\alpha) + \eta(\beta) \\[6pt] \eta(\lambda\alpha) &= \lambda\eta(\alpha) \end{aligned} η(α+β)η(λα)​=η(α)+η(β)=λη(α)​

则线性空间 V1 ~V_1~ V1​ 与 V2 ~V_2~ V2​ 称为同构的，而映射 η ~\eta~ η 称为 V1 ~V_1~ V1​ 到 V2 ~V_2~ V2​ 上的一个同构映射。

满足这两个等式的映射 η ~\eta~ η 称为保线性关系的。它也有自反性，对称性和传递性，说明这对线性空间进行了一种分类。实际上，可以证明，数域 F ~\mathbb{F}~ F 上的任意一个 n ~n~ n 维线性空间 V ~V~ V 都同构于数域 F ~\mathbb{F}~ F 上的 n ~n~ n 维向量空间 Fn ~\mathbb{F}^n~ Fn .

如以此来，尽管我们在定义线性空间的时候并没有对元素的具体内容做任何的要求，但线性空间必然同构于一个向量空间，那么线性空间上的计算便可以转化为对应的向量空间上的计算。由于向量空间的元素都是 n ~n~ n 元有序数组，这样线性空间的各种运算便简便了很多。

至此，线性空间的介绍先告一段落。我们发现，线性空间的概念是相当广泛的。实际上，本应是由求解线性方程组而引出线性空间，但线性空间的概念是如此的有用，可以帮助我们理解很多线性代数的内容。我们便将其作为本篇总结的开头。接下来，会慢慢引出线性代数中的其他概念，并简单的展示理解线性空间的好处。

行列式

n 阶行列式的定义与性质

 n ~n~ n 维向量空间 Fn ~\mathbb{F}^n~ Fn 上的规范反对称 n ~n~ n 重线性函数称为数域 F ~\mathbb{F}~ F 上的 n ~n~ n 阶行列式函数，简称为 n ~n~ n 阶行列式。

“规范”表明单位阵的行列式为1；“反对称”表明交换行列式两行的位置，行列式变号；“ n ~n~ n 重线性”表明行列式对每一行都具有线性性；“函数”表明行列式的结果为一个值。

由定义，可以导出以下性质

  • 对换 n ~n~ n 阶行列式的某两行，其值变号；
  •  n ~n~ n 阶行列式的某一行遍乘以某一个数，加到另一行，其值不变；
  •  n ~n~ n 阶行列式的某一行遍乘以数 λ ~\lambda~ λ , 其值为原行列式的 λ ~\lambda~ λ 倍。

行列式的定义看似有些莫名其妙，实际上是定义了 n ~n~ n 维向量空间中的某种有向体积（对于二维应当称为面积）。我们知道，如果把行列式的行（或列）分别看作向量，则二阶行列式表示平面中某个平行四边形的面积，三阶行列式表示空间中某个平行六面体的体积。

以三阶行列式为例，对换行列式的某两行，相当于将坐标系由右手系换成了左手系（或左手系换成了右手系），由于行列式是有向体积，故值变号；行列式两行的线性组合，相当于把构成体积的某一条边进行平移，相应的，将行列式化为对角形式就相当于通过平移使各条边与单位正交基平行；将行列式的某行乘一个常数，就相当于把平行六面体的某条边扩大相应的倍数，体积自然要扩大相应的倍数。

以上关于体积的关系也可以推广到 n ~n~ n 维向量空间。 n ~n~ n 维空间的体积我们可能难以想象，而行列式恰好给我们提供了一种很好的计算途径。

另外，对于行列式而言，行和列的地位是相当的，也就是说，行列式转置后值不变。

由行列式的几何意义，可以知道，当行列式某行为零（某条边长度为零）或某两行仅相差一个倍数（两条边共线）或某一行为另两行的线性组合（某三条边共面）的时候，行列式的值为零（体积为零）。

行列式的计算

计算技巧并非本文的重点，详细内容可以参考相关教材，本节只列出一些基本的操作方法。

计算行列式的基本思想是通过行列式的初等变换来使行列式的形式变得更加简单。

对行列式实施的初等变换是指

  • 对换行列式的某两行（列），其它的行（列）保持不动；
  • 行列式的某一行（列）的元素遍乘以某个非零的数再加到另一行（列）；
  • 行列式的某一行（列）的元素遍乘以某个非零的数。

基本方法有如下几种

  • 化为三角形
  • 建立递推公式
  •  Laplace ~\mathrm{Laplace}~ Laplace 展开
  • 加边
  • 拆行（列）
  • 视行列式的某些元素为未知数

矩阵

定义及含义

矩阵是 m×n ~m\times n~ m×n 个数排列成的一个长方形表，它有 m ~m~ m 行， n ~n~ n 列。 m=n ~m = n~ m=n 时称为 n ~n~ n 阶方阵。

矩阵的加法即为对应元素的加法，数乘即所有元素乘以相应的数。可以验证，矩阵的加法和数乘满足前面给过的八条公理，所有我们完全有理由把 m×n ~m\times n~ m×n 阶矩阵全体称为数域 F ~\mathbb{F}~ F 上的 m×n ~m\times n~ m×n 维向量空间。与前面向量的定义不同的是，矩阵之间还引进了乘法。

可以发现，矩阵 C ~C~ C 的每一行都是 B ~B~ B 的行的线性组合，矩阵 C ~C~ C 的每一列都是 A ~A~ A 的列的线性组合。将矩阵 A ~A~ A 的第 i ~i~ i 行乘以 B ~B~ B 的第 j ~j~ j 列得到 cij ~c_{ij}~ cij​ ，这样可以得到矩阵的每一个元素。而将矩阵 A ~A~ A 的第 i ~i~ i 列乘以 B ~B~ B 的第 j ~j~ j 行可以得到一个矩阵，矩阵 C ~C~ C 是所有这些矩阵的线性组合。

实际上矩阵乘法的定义是按照线性映射来的。后面将会看到，矩阵的乘法与线性映射天然的吻合，那就是因为矩阵乘法就是为了表达一种线性映射而定义的。这样也可以看出，矩阵的意义实际上是一种线性操作，它对线性空间中的一些向量进行了线性组合。

容易验证，矩阵乘法有结合律，而一般不具有交换律。不难想到，线性映射的结果与操作顺序有关。

矩阵可以进行分块运算，每一块仍然按照矩阵的性质来进行运算。

了解了矩阵实际上可以看作线性映射操作，那么关于矩阵的一些运算性质实际上就很好理解了。

秩与相抵

定义 m×n ~m\times n~ m×n 矩阵 A ~A~ A 的所有非零子式的最高阶数称为矩阵 A ~A~ A 的秩，记为 rankA ~\mathrm{rank}A~ rankA 。

可以通过初等变换来求矩阵的秩。初等行变换相当于左乘可逆矩阵，初等列变换相当于右乘可逆矩阵。

将一个矩阵的每一行都视为向量，则由矩阵可以得到一个行向量空间。矩阵的行秩，给出了行向量空间的维数，所对应的行，就是行向量空间的一组基。相应的矩阵的列也可以有同样的看法。并且可以证明，矩阵的行秩和列秩是相等的。

设 A, B∈Fm×n ~A,~B\in\mathbb{F}^{m\times n}~ A, B∈Fm×n 。如果矩阵 A ~A~ A 可以经过有限次初等变换变为矩阵 B ~B~ B ，则矩阵 A ~A~ A 和 B ~B~ B 是相抵的。

或简记为

PAQ=BPAQ = B PAQ=B

矩阵的相抵也具有自反性，对称性和传递性，说明相抵也是一种分类，相抵的矩阵属于同一类。

既然相抵的矩阵是同一类的，那我们不妨给相抵找一个代表元，那就是 Hermite ~\mathrm{Hermite}~ Hermite 标准型。

这也表明，矩阵的秩是矩阵在相抵下的不变量，而且是全系不变量。

相抵的要求相当宽松。实际上，任何一个矩阵都可以分解为一个列满秩矩阵与一个行满秩矩阵的乘积。这一事实称为矩阵的满秩分解定理。

线性方程组

将线性方程组整齐排列，依照矩阵乘法的定义，可以将其记为 AX=β ~AX=\beta~ AX=β 。其中 A ~A~ A 为系数矩阵， X ~X~ X 为未知数组成的矩阵， β ~\beta~ β 为一个列向量。 β ~\beta~ β 不为零向量时，方程组称为非齐次的，反之称为齐次的。

求解线性方程组的基本方法是消元。利用 Gauss ~\mathrm{Gauss}~ Gauss 消元法求解线性方程组，仅使用了三种基本变换：

  • 互换矩阵 A ~A~ A 的两行；
  • 将矩阵 A ~A~ A 的某行乘一个非零常数；
  • 将矩阵 A ~A~ A 的某行乘一个非零常数加到另一行.

这被称为矩阵的三个初等变换。

通过初等变换，将矩阵 A ~A~ A 化为一种阶梯状。剩下的行线性无关，行数便是矩阵 A ~A~ A 的秩。

线性方程组的解集合常常不止一个元素，将它们写为一些变元线性组合的形式，这称为通解。当变元取定特定的值，得到一个特解。不难看出，齐次线性方程组总有一个平凡解，即零解。

子空间

在三维空间中，我们知道一条过原点的直线上的向量对加法和数乘是封闭的，过原点的平面上的向量也有同样的结果。我们现在将这种性质抽象出来，并扩展到 n ~n~ n 维空间中。这就构成了子空间的概念。

定义 U ~U~ U 是数域 F ~\mathbb{F}~ F 上的线性空间 V ~V~ V 的非空向量集合。如果 U ~U~ U 对于线性空间 V ~V~ V 的向量加法和纯量与向量的乘法是封闭的，即对任意 αβ∈U ~\alpha\beta\in U~ αβ∈U ，有  α+β∈U  ~~\alpha + \beta\in U~~  α+β∈U  ，并且对任意 λ∈F, α∈U ~\lambda\in\mathbb{F},~\alpha\in U~ λ∈F, α∈U ，有 λα∈U ~\lambda\alpha\in U~ λα∈U ，则 U ~U~ U 称为线性空间 V ~V~ V 的子空间。

简单的讲，子空间就是对向量加法和数乘封闭的非零向量集合。由定义可知，子空间里一定包含零向量。仅包含零向量的子空间称为零子空间，记为 O ~O~ O .

我们很自然的想到，前面齐次线性方程组的解一定包含一个平凡解。那么齐次线性方程组的解集合是否构成一个子空间呢？答案是肯定的。

我们可以将齐次线性方程组的系数矩阵视为由列向量排列成的矩阵，那么求解齐次线性方程组就是找到一组组合系数，使这些列向量组合出零向量。若这些列向量是线性无关的（此时矩阵满秩），那么方程仅有零解；若是线性相关的，那么方程有无穷多组解。

我们不妨把解集合中的元素也视为向量。可以验证，它们所构成的空间是线性空间，因为解的和以及常数倍仍然是原方程的解。不难看出，解空间的维数取决于系数矩阵的秩。实际上有如下关系

dim⁡VA=n − rankA\dim V_A = n ~-~ \mathrm{rank}A dimVA​=n − rankA

其中 VA ~V_A~ VA​ 为解空间。解空间的一组基称为基础解系。

从几何上看，解空间必定经过原点。

对于线性非齐次方程组 AX=β ~AX = \beta~ AX=β ，可以看出，齐次方程组的通解加上任意一个非齐次方程组的特解都满足非齐次方程组。这便是线性方程组的解结构定理。

可见，既然齐次线性方程组的解空间是经过原点的子空间，那么非齐次方程组的解空间就是其平移过后的结果。它并不构成一个子空间，但它可以由相应的子空间得到。

下面介绍子空间的相关运算。

  • 任意多个子空间的交是子空间

设 V1 ~V_1~ V1​ 和 V2 ~V_2~ V2​ 是数域 F ~\mathbb{F}~ F 上的线性空间 V ~V~ V 的子空间，则集合

  • 线性空间 V ~V~ V 的子空间 V1 ~V_1~ V1​ 与 V2 ~V_2~ V2​ 的和 V1+V2 ~V_1+V_2~ V1​+V2​ 是 V ~V~ V 的子空间。

  • 维数公式

    dim⁡V1+dim⁡V2=dim⁡(V1∩V2)+dim⁡(V1+V2)\dim V_1 + \dim V_2 = \dim(V_1 \cap V_2) + \dim(V_1 + V_2) dimV1​+dimV2​=dim(V1​∩V2​)+dim(V1​+V2​)

这意味着两个子空间的交越多，和就越少。

在子空间中，由向量集合生成的子空间具有重要的作用。

商空间

设 V ~V~ V 是数域 F ~\mathbb{F}~ F 上的有限维线性空间 V ~V~ V 的子空间，向量 α,β∈V ~\alpha,\beta\in V~ α,β∈V .如果向量 α−β∈W ~\alpha - \beta \in W~ α−β∈W ，则向量 α ~\alpha~ α 与 β ~\beta~ β 称为模 W ~W~ W 同余，记为 α≡β (mod W) ~\alpha \equiv \beta~(\mathrm{mod}~W)~ α≡β (mod W) .

α‾=α+W={α+γ∣γ∈W}\overline\alpha = \alpha + W = \{ \alpha + \gamma \mid \gamma \in W \} α=α+W={α+γ∣γ∈W}

应当指出，上面规定的同余类加法是在同余类 α‾ ~\overline\alpha~ α 与 β‾ ~\overline\beta~ β​ 中各取出一个向量 α ~\alpha~ α 与 β ~\beta~ β （它们分别称为同余类 α‾ ~\overline\alpha~ α 与 β‾ ~\overline\beta~ β​ 的代表元），然后用和 α+β ~\alpha + \beta~ α+β 所在的同余类 α+β‾ ~\overline{\alpha + \beta}~ α+β​ 作为同余类 α‾ ~\overline\alpha~ α 与 β‾ ~\overline\beta~ β​ 的和 α‾+β‾ ~\overline\alpha + \overline\beta~ α+β​ ．可以证明，选取不同的代表元得到的结果是相同的。

映射

本节记录一些基本概念。

给定集合 S ~S~ S 和 T ~T~ T 。所谓集合 S ~S~ S 到 T ~T~ T 的映射 η ~\eta~ η 是指集合 S ~S~ S 到集合 T ~T~ T 的一个对应规律，使得对任意给定的元素 α∈S ~\alpha\in S~ α∈S ，按照对应规律 η ~\eta~ η ，可以确定集合 T ~T~ T 的唯一一个元素 β ~\beta~ β 与 α ~\alpha~ α 相对应。

元素 β ~\beta~ β 称为元素 α ~\alpha~ α 在映射 η ~\eta~ η 下的象，元素 α ~\alpha~ α 称为元素 β ~\beta~ β 的原象．当 α ~\alpha~ α 遍历集合 S ~S~ S 的元素时，所有 α ~\alpha~ α 的象的集合称为集合 S ~S~ S 在映射 η ~\eta~ η 下的象，记为 Im η ~\mathrm{Im}~ \eta~ Im η ，或者 η(S) ~\eta(S)~ η(S) . 即

Im η={η(α)∣α∈S}\mathrm{Im}~\eta = \{\eta(\alpha) \mid \alpha \in S\} Im η={η(α)∣α∈S}

  • 给定映射 η:S→T ~\eta : S \to T~ η:S→T . 如果对任意 α1α2∈S,α1≠α2 ~\alpha_1 \alpha_2 \in S,\alpha_1\neq\alpha_2~ α1​α2​∈S,α1​​=α2​ ，均有 η(α1)≠η(α2) ~\eta(\alpha_1)\neq\eta(\alpha_2)~ η(α1​)​=η(α2​) ，则 η ~\eta~ η 称为单射。可以看出，当且仅当由 η(α1)=η(α2) ~\eta(\alpha_1)=\eta(\alpha_2)~ η(α1​)=η(α2​) 可以推出 α1=α2 ~\alpha_1 = \alpha_2~ α1​=α2​ 时， η ~\eta~ η 是单射。

  • 给定映射 η:S→T ~\eta: S \to T~ η:S→T . 如果对任意 β∈T ~\beta\in T~ β∈T ，存在 α∈S ~\alpha\in S~ α∈S ，使得 η(α)=β ~\eta(\alpha)=\beta~ η(α)=β ，则 η ~\eta~ η 称为满射。容易看出，当且仅当 η(S)=T ~\eta(S) = T~ η(S)=T 时，映射 η ~\eta~ η 是满射。如果 η ~\eta~ η 是满射，则说映射 η ~\eta~ η 是 S ~S~ S 到 T ~T~ T 上的；如果 η ~\eta~ η 不是满射，则说映射 η ~\eta~ η 是 S ~S~ S 到 T ~T~ T 内的。

  • 如果映射 η:S→T ~\eta : S \to T~ η:S→T 既是单射，又是满射，则 η ~\eta~ η 称为双射，或者 S ~S~ S 到 T ~T~ T 上的一一对应。

给定集合 U,V ~U,V~ U,V 和 W ~W~ W ，以及映射 η:U→V ~\eta: U \to V~ η:U→V ， ξ:V→W ~ξ : V \to W~ ξ:V→W ，定义 U ~U~ U 到 W ~W~ W 的合成映射 ξη ~\xi\eta~ ξη 如下：设 u∈U ~u \in U~ u∈U ，则令

(ξη)(u)=ξ(η(u))(\xi\eta)(u) = \xi(\eta(u)) (ξη)(u)=ξ(η(u))

映射 ξη:U→W ~\xi\eta : U \to W~ ξη:U→W 称为映射 η ~\eta~ η 和 ξ ~\xi~ ξ 的乘积。

  • 映射 η:S→T ~\eta : S \to T~ η:S→T 可逆的充分必要条件是映射 η ~\eta~ η 为双射。

  • 设映射 η:S→T ~\eta : S \to T~ η:S→T 可逆，则 η ~\eta~ η 的逆映射是唯一的。

  • 设映射 η:S→T ~\eta : S \to T~ η:S→T 和 ξ:T→U ~\xi : T \to U~ ξ:T→U 可逆，则它们的乘积 ξη:S→U ~\xi\eta : S \to U~ ξη:S→U 也可逆。

不难看出，映射的一些性质与矩阵似乎有某种相似之处。实际上，我们可以认为矩阵就是根据接下来介绍的线性映射来定义的。

线性映射

映射描述了任意两个集合之间的对应关系。我们可以把这种对应关系运用到两个线性空间上。

设 U ~U~ U 和 V ~V~ V 是数域 F ~\mathbb F~ F 上的线性空间，且 A:U→V ~A : U \to V~ A:U→V 是映射。

如果映射 A:U→V ~\mathscr{A} : U \to V~ A:U→V 满足：

  • 对任意 α1,α2∈U ~\alpha_1,\alpha_2\in U~ α1​,α2​∈U ，有

A(α1+α2)=A(α1)+A(α2)\mathscr{A}(\alpha_1 + \alpha_2) = \mathscr{A}(\alpha_1) + \mathscr{A}(\alpha_2) A(α1​+α2​)=A(α1​)+A(α2​)

  • 对任意 λ∈F,α∈U ~\lambda\in\mathbb{F},\alpha\in U~ λ∈F,α∈U ，有

A(λα)=λA(α)\mathscr{A}(\lambda \alpha) = \lambda\mathscr{A}(\alpha) A(λα)=λA(α)

则映射 A ~\mathscr{A}~ A 称为线性空间 U ~U~ U 到 V ~V~ V 的线性映射。

可以证明线性映射有如下性质

  • 设 0 ~\boldsymbol 0~ 0 是线性空间 U ~U~ U 的零向量，则 A(0)=0 ~\mathscr{A}(\boldsymbol 0)=\boldsymbol 0~ A(0)=0 .

  • 设 λ1,λ2,…,λk∈F,α1,α2,…,αk∈U ~\lambda_1,\lambda_2,\ldots,\lambda_k\in\mathbb{F},\alpha_1,\alpha_2,\ldots,\alpha_k \in U~ λ1​,λ2​,…,λk​∈F,α1​,α2​,…,αk​∈U ，则

    A(λ1α1+λ2α2+⋯+λkαk)=λ1A(α1)+λ2A(α2)+⋯+λkA(αk)\mathscr{A}(\lambda_1\alpha_1 + \lambda_2\alpha_2 + \cdots + \lambda_k\alpha_k) = \lambda_1\mathscr{A}(\alpha_1) + \lambda_2\mathscr{A}(\alpha_2) + \cdots + \lambda_k\mathscr{A}(\alpha_k) A(λ1​α1​+λ2​α2​+⋯+λk​αk​)=λ1​A(α1​)+λ2​A(α2​)+⋯+λk​A(αk​)

  • 设向量组 α1,α2,…,αk∈U ~\alpha_1,\alpha_2,\ldots,\alpha_k\in U~ α1​,α2​,…,αk​∈U 线性相关，则 λ1A(α1),…,λkA(αk)∈V ~\lambda_1\mathscr{A}(\alpha_1),\ldots,\lambda_k\mathscr{A}(\alpha_k)\in V~ λ1​A(α1​),…,λk​A(αk​)∈V 也线性相关。

  • 设 U ~U~ U 与 V ~V~ V 分别是数域 F ~\mathbb F~ F 上 n ~n~ n 维和 m ~m~ m 维线性空间， {α1,α2,…,αn} ~\{\alpha_1,\alpha_2,\ldots,\alpha_n\}~ {α1​,α2​,…,αn​} 是 U ~U~ U 的基。设 β1,β2,…,βn ~\beta_1,\beta_2,\ldots,\beta_n~ β1​,β2​,…,βn​ 是 V ~V~ V 中任意给定的 n ~n~ n 个向量。则存在唯一线性映射 A:U→V ~\mathscr A : U \to V~ A:U→V ，使得 A(αj)=βj,j=1,2,…,n ~\mathscr A(\alpha j) = \beta j,j=1,2,\ldots,n~ A(αj)=βj,j=1,2,…,n .

    这表明，如果在线性空间 U ~U~ U 中取定一组基 {α1,α2,…,αn} ~\{\alpha_1,\alpha_2,\ldots,\alpha_n\}~ {α1​,α2​,…,αn​} ，则线性映射 A:U→V ~\mathscr A : U \to V~ A:U→V 由基向量 α1,α2,…,αn ~\alpha_1,\alpha_2,\ldots,\alpha_n~ α1​,α2​,…,αn​ 在 A ~\mathscr A~ A 下的象所唯一决定。

即

矩阵 A ~A~ A 便称为线性映射 A ~\mathscr{A}~ A 在 U ~U~ U 的基 {α1,α2,…,αn} ~\{\alpha_1,\alpha_2,\ldots,\alpha_n\}~ {α1​,α2​,…,αn​} 与 V ~V~ V 的基 {β1,β2,…,βn} ~\{\beta_1, \beta_2, \ldots, \beta_n\}~ {β1​,β2​,…,βn​} 下的矩阵。

设 α∈U ~\alpha\in U~ α∈U 在基 {α1,α2,…,αn} ~\{\alpha_1,\alpha_2,\ldots,\alpha_n\}~ {α1​,α2​,…,αn​} 下的坐标为 xT=(x1,x2,…,xn) ~x^T = (x_1,x_2,\ldots,x_n)~ xT=(x1​,x2​,…,xn​) ，则

A(α)=(β1,β2,…,βm)Ax\mathscr A(\alpha) = (\beta_1, \beta_2, \ldots, \beta_m)Ax A(α)=(β1​,β2​,…,βm​)Ax

这表明向量 α ~\alpha~ α 在 A ~\mathscr A~ A 下的象 A(α) ~\mathscr A(\alpha)~ A(α) 在 V ~V~ V 的基 {β1,β2,…,βn} ~\{\beta_1, \beta_2, \ldots, \beta_n\}~ {β1​,β2​,…,βn​} 下的坐标 y ~y~ y 为

y=Axy = Ax y=Ax

由上面的叙述我们可以看出，数域 F ~\mathbb{F}~ F 上一个 m×n ~m \times n~ m×n 矩阵表示数域 F ~\mathbb{F}~ F 上 n ~n~ n 维线性空间 U ~U~ U 到数域 F ~\mathbb{F}~ F 上 m ~m~ m 维线性空间 V ~V~ V 的一个线性映射。
而相抵的矩阵，就是同一个线性映射在不同基下的矩阵。这也表明，相抵所作的分类实际上是给线性映射进行了分类。

既然矩阵对应着线性映射，那么矩阵乘法实际上就对应了线性映射操作。用矩阵来乘一个向量（若是矩阵，视为多行向量），就是对向量进行线性映射，得到的结果将是该向量的象。

线性映射的代数运算

线性映射之间的代数运算也具有线性性。这是因为线性映射同构于一个矩阵，而矩阵的代数运算是满足线性性的。并且，记 U ~U~ U 到 V ~V~ V 的所有线性映射为 Lm×n ~\mathscr{L}_{m \times n}~ Lm×n​ ，则容易验证，集合 Lm×n ~\mathscr{L}_{m \times n}~ Lm×n​ 在加法和纯量与映射的乘法下构成数域 F ~\mathbb{F}~ F 上的 m×n ~m \times n~ m×n 维线性空间。且

dim⁡Lm×n=mn\dim \mathscr{L}_{m \times n} = m n dimLm×n​=mn

对于可逆的线性映射，有如下定理。

  • 设线性 A:U→V ~A : U \to V~ A:U→V 可逆，则 dim⁡U=dim⁡V ~\dim U = \dim V~ dimU=dimV .
    这表明，如果线性映射是可逆的，则两个空间是同构的。

  • 设 A:U→V ~A : U \to V~ A:U→V 与 B:V→W ~B : V \to W~ B:V→W 是可逆线性映射。则乘积 BA:U→W ~BA : U \to W~ BA:U→W 是可逆线性映射。
    这说明可以逆映射的符合仍然是可逆的。同时也表明可逆矩阵的乘积仍然是可逆矩阵。

象与核

前面已经提到过象的概念，这里给出定义。

设 A:U→V ~A : U \to V~ A:U→V 是线性映射。则集合

Im A={A(α)∈V∣α∈U}\mathrm{Im}~\mathscr{A} = \{\mathscr{A}(\alpha)\in V \mid \alpha \in U\} Im A={A(α)∈V∣α∈U}

称为 U ~U~ U 在 A ~\mathscr{A}~ A 下的象，或者 A ~\mathscr{A}~ A 的值域。也记为 A(U) ~\mathscr{A}(U)~ A(U) .
集合

Ker A={α∈U∣A(α)=0∈V}\mathrm{Ker}~\mathscr{A} = \{\alpha \in U \mid \mathscr{A}(\alpha) = 0 \in V\} Ker A={α∈U∣A(α)=0∈V}

称为 A ~\mathscr{A}~ A 的核，也记为 A−1(0) ~\mathscr{A}^{-1}(0)~ A−1(0) .

简单的讲，象就是映射过来的结果，核就是映射到零向量的原象。
有了象与核的概念，描述一些问题会更加简便。

接下来有如下定理

  • 设 A:U→V ~A : U \to V~ A:U→V 是线性映射，则 U ~U~ U 在 A ~A~ A 下的象 Im A ~\mathrm{Im}~A~ Im A 是 V ~V~ V 的子空间，而 A ~A~ A 的核 Ker A ~\mathrm{Ker}~A~ Ker A 是 U ~U~ U 的子空间。

  • 设 A:U→V ~A : U \to V~ A:U→V 是线性映射，则 U ~U~ U 模 Ker A ~\mathrm{Ker}~A~ Ker A 的商空间 U/Ker A ~U/\mathrm{Ker}~A~ U/Ker A 同构于 Im A ~\mathrm{Im}~A~ Im A .
    这表示，取模把那些无关的向量都商掉了，说明商空间有一种去除无关向量的能力。具体的讲，核中的向量由于映射到零向量，所以与其他向量相加之后不起作用，故商空间将它们全部去掉之后就只剩下了核外的向量。此时的原象与象有着一一对应的关系，便构成的同构映射。

  • 设 A:U→V ~A : U \to V~ A:U→V 是线性映射，则

    dim⁡U=dim⁡Im A+dim⁡Ker A\dim U = \dim \mathrm{Im}~A + \dim \mathrm{Ker}~A dimU=dimIm A+dimKer A

    这说明 U ~U~ U 中的向量被分为两部分：映射到零向量的向量和映射到非零向量的向量。
    这看似很简单很自然，但也是一种有用的分类。

设 A:U→V ~A : U \to V~ A:U→V 是线性映射，则 dim⁡Im A ~\dim\mathrm{Im}~A~ dimIm A 称为 A ~A~ A 的秩，记为 ρ(A) ~\rho(A)~ ρ(A) ；而 dim⁡Ker A ~\dim \mathrm{Ker}~A~ dimKer A 称为 A ~A~ A 的零度，记为 ν(A) ~\nu(A)~ ν(A) .

这是从线性映射的观点用象与核的概念重新描述了秩与零空间。

前面讲过矩阵可以相抵于一个对角阵。对角元为一的位置即是象的所在，对角元为零的位置即是核的所在。而我们知道，矩阵的秩，也就是 Hermite ~\mathrm{Hermite}~ Hermite 标准型的秩是相抵的全系不变量。这就说明把相抵的矩阵作为线性映射对应的矩阵去作用时，象与核的秩都是不变的，它们都是同构的，是同一个线性映射，不同的仅仅是组合的向量。这就说明，相抵的矩阵是同一个线性映射在不同基下的矩阵。

线性变换

线性映射描述的是集合 U→V ~U \to V~ U→V 的映射，当集合映射到自身时，我们通常称其为线性变换。实际上，在本次线性代数学习中，线性变换应当是重点内容。

由上一节我们可以知道，矩阵 A ~A~ A 一定是可逆的。

设 {β1,β2,…,βn} ~\{\beta_1,\beta_2,\ldots,\beta_n\}~ {β1​,β2​,…,βn​} 是 V ~V~ V 的另一组基，线性变换 A:V→V ~\mathscr{A} : V \to V~ A:V→V 在基 {β1,β2,…,βn} ~\{\beta_1,\beta_2,\ldots,\beta_n\}~ {β1​,β2​,…,βn​} 下的方阵为 B ~B~ B ，即

A(β1,β2,…,βn)=(β1,β2,…,βn)B\mathscr{A}(\beta_1,\beta_2,\ldots,\beta_n) = (\beta_1,\beta_2,\ldots,\beta_n)B A(β1​,β2​,…,βn​)=(β1​,β2​,…,βn​)B

设基 {α1,α2,…,αn} ~\{\alpha_1,\alpha_2,\ldots,\alpha_n\}~ {α1​,α2​,…,αn​} 到基 {β1,β2,…,βn} ~\{\beta_1,\beta_2,\ldots,\beta_n\}~ {β1​,β2​,…,βn​} 的过渡矩阵为 P ~P~ P 。则可以推出 B=P−1AP ~B = P^{-1}AP~ B=P−1AP .我们把满足这种关系的方阵 A,B ~A, B~ A,B 称为相似的。

由上面的过程可知，相似的矩阵是同一个线性变换在不同基下的方阵。

与相抵类似，相似也有自反性，对称性，传递性。这说明我们可以利用相似关系来对方阵进行分类，相似的方阵属于同一类。
分类后，我们总是希望给同一类的元素找一个代表元。在相抵中，我们找的是 Hermite ~\mathrm{Hermite}~ Hermite 矩阵，这是因为我们很自然的希望代表元越简单越好。那么在相似中，我们也希望能够给所有的类找到一个最简单的代表元。我们很自然的会想到对角阵，也就是说我们希望通过选择一组合适的基来把一个矩阵相似到对角阵。但遗憾的是并非所有矩阵都能相似到对角阵。对于实方阵我们只能相似到 Jordan ~\mathrm{Jordan}~ Jordan 标准型组成的矩阵。这些内容将在后面一一展示出来。

特征值与特征向量

线性变换是可逆的，那么对基 {α1,α2,…,αn} ~\{\alpha_1,\alpha_2,\ldots,\alpha_n\}~ {α1​,α2​,…,αn​} 进行的线性变换将会得到象的一组基 {A(α1),A(α2),…,A(αn)} ~\{\mathscr{A}(\alpha_1),\mathscr{A}(\alpha_2),\ldots,\mathscr{A}(\alpha_n)\}~ {A(α1​),A(α2​),…,A(αn​)} .
但是既然象也是在同一个空间中，我们为什么不选同一组基呢？那么翻译成数学语言，就是希望对于基向量 α ~\alpha~ α ，有 A(α)=α ~\mathscr{A}(\alpha) = \alpha~ A(α)=α 但是这样未必办得到。比如我们将基向量做旋转变换，若非旋转 2π ~2\pi~ 2π 的整数倍，得到的向量是不会与原来的向量重合的。那么我们退而求其次，希望对于基向量 α ~\alpha~ α ，有 A(α)=λα ~\mathscr{A}(\alpha) = \lambda\alpha~ A(α)=λα ，其中 λ ~\lambda~ λ 为任意常数。这就是希望基向量经过变换后与原来共线。

设 A:V→V ~A : V \to V~ A:V→V 是线性变换。如果存在非零向量 α∈V ~\alpha\in V~ α∈V ，以及纯量 λ∈F ~\lambda\in\mathbb{F}~ λ∈F ，使得

A(α)=λα\mathscr{A}(\alpha) = \lambda\alpha A(α)=λα

则 λ ~\lambda~ λ 称为线性变换 A ~A~ A 的特征值， α ~\alpha~ α 称为线性变换 A ~A~ A 的属于特征值 λ ~\lambda~ λ 的特征向量。

那么我们既要寻找特征值 λ ~\lambda~ λ ，又要寻找它所对应的特征向量 α ~\alpha~ α .

我们首先将基向量 α ~\alpha~ α 视为变量，会发现这是一个齐次方程

(λI−A)α=0(\lambda I - A)\alpha = 0 (λI−A)α=0

要使得 α ~\alpha~ α 有非零解，必须有

det⁡(λI−A)=0\det(\lambda I - A) = 0 det(λI−A)=0

现在把 λ ~\lambda~ λ 视为变量，那么行列式展开得到一个关于 λ ~\lambda~ λ 的多项式，我们称之为特征多项式。特征值 λ ~\lambda~ λ 即是特征多项式的根。解出所有特征值后，再进行回代解出所有的 α ~\alpha~ α ，即可得到所有的特征向量。

我们选择一组合适的基，如果这些基都有上述关系的话，方阵 A ~A~ A 就是可以相似到对角阵的，否则不行。

一个空间进行线性变换时，总有一些东西是不变或近乎不变的。比如将一个圆拉伸为椭圆，沿着拉伸的方向的向量在变换前后是共线的。这个向量就是特征向量，拉伸或压缩的倍数就是特征值。
特征向量有时候是可见的，有时候是不可见的。比如三维空间中旋转的转轴就是特征向量（且特征值为 1 ~1~ 1 ），它在旋转变换中是不变的；而在二维空间中的旋转我们虽然知道有转轴，但是转轴并不在这个二维空间中，这是因为这时特征值为复数，既然是复数，当然在实的空间中就不可见了。

下面我们指出，如果方阵 A,B ~A,B~ A,B 相似，则它们的特征多项式相同，从而特征值相同。这说明，特征值和特征多项式是相似下的不变量。

接下来我们对特征多项式稍做研究。
特征多项式也是多项式，所以在复数范围内它一定可以分解为若干一次式

φA(λ)=λn+a1λn−1+⋯+an−1λ+an=(λ−λ1)(λ−λ2)⋯(λ−λn)\begin{aligned} \varphi_A(\lambda) &= \lambda^n + a_1\lambda^{n-1} + \cdots + a_{n-1}\lambda + a_n \\ &= (\lambda-\lambda_1)(\lambda-\lambda_2)\cdots(\lambda-\lambda_n) \end{aligned} φA​(λ)​=λn+a1​λn−1+⋯+an−1​λ+an​=(λ−λ1​)(λ−λ2​)⋯(λ−λn​)​

则有

tr A=−a1=λ1+λ2+⋯+λndet⁡A=(−1)nan=λ1λ2⋯λn\begin{aligned} \mathrm{tr}~A &= -a_1 = \lambda_1 + \lambda_2 + \cdots + \lambda_n \\ \det A &= (-1)^n a_n = \lambda_1\lambda_2\cdots\lambda_n \end{aligned} tr AdetA​=−a1​=λ1​+λ2​+⋯+λn​=(−1)nan​=λ1​λ2​⋯λn​​

在特征多项式分解成的一次式中，可能会有重根，那么应当写为如下形式

φA(λ)=(λ−λ1)n1(λ−λ2)n2⋯(λ−λl)nl\varphi_A(\lambda) = (\lambda-\lambda_1)^{n_1}(\lambda-\lambda_2)^{n_2}\cdots(\lambda-\lambda_l)^{n_l} φA​(λ)=(λ−λ1​)n1​(λ−λ2​)n2​⋯(λ−λl​)nl​

其中每个因子 λ−λi ~\lambda - \lambda_i~ λ−λi​ 的次数 ni ~n_i~ ni​ 为该特征值的代数重数。代数重数至少为一，且所有特征值的代数重数之和为 n ~n~ n .

由上面的叙述我们可以知道，如果两个方阵 A,B ~A,B~ A,B 是相似的，那么它们的迹和行列式的值都是相等的。

特征子空间

对于方阵 A ~A~ A 的任意一个特征值 λ0 ~\lambda_0~ λ0​ ，齐次线性方程组

(A−λ0I)X=0(A - \lambda_0 I)X = \boldsymbol 0 (A−λ0​I)X=0

的解空间 Vλ0=Ker (A−λ0I) ~V_{\lambda_0} = \mathrm{Ker}~(A - \lambda_0 I)~ Vλ0​​=Ker (A−λ0​I) 不为零，其维数大于等于一。 Vλ0 ~V_{\lambda_0}~ Vλ0​​ 中所有非零向量就是属于特征值 λ0 ~\lambda_0~ λ0​ 的全部特征向量。
