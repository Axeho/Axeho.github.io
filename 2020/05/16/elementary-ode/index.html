常微分方程初步
Created 2020-05-16| Updated 2020-05-28|学习总结
Post View:|Comments:

本文记录了一些基本的常微分方程的解法。写文章的主要目的是方便作者以后的复习。欢迎在评论区与本人交流。鉴于作者水平有限，如若内容有误，还请读者及时批评指正。

一阶微分方程

分离变量法

形如

dydx=f(x,y)\frac{\mathrm{d}y}{\mathrm{d}x} = f(x,y) dxdy​=f(x,y)

若 f(x,y)=g(x)h(y) ~f(x,y) = g(x)h(y)~ f(x,y)=g(x)h(y) ，这里 g,h ~g,h~ g,h 分别是 x ~x~ x 和 y ~y~ y 的连续函数，且 h(y) ~h(y)~ h(y) 不恒为零。则方程在 h(y)≠0 ~h(y)\ne 0~ h(y)​=0 时可化为

dyh(y)=g(x)dx\frac{\mathrm{d}y}{h(y)} = g(x)\mathrm{d}x h(y)dy​=g(x)dx

积分可得

∫dyh(y)=∫g(x) dx\int \frac{\mathrm{d}y}{h(y)} = \int g(x)~\mathrm{d}x ∫h(y)dy​=∫g(x) dx

即

H(y)=G(x)+CH(y) = G(x) + C H(y)=G(x)+C

齐次方程

有些方程本身不能直接分离变量，但作适当的代换后，可用分离变量法求解。

一个函数 f(x,y) ~f(x,y)~ f(x,y) 称为 n ~n~ n 次齐次函数，如果对某个范围内的 x, y ~x,~y~ x, y 与 t ~t~ t 有

f(tx,ty)=tnf(x,y)f(tx,ty) = t^nf(x,y) f(tx,ty)=tnf(x,y)

一阶微分方程

P(x,y)dx+Q(x,y)dy=0P(x,y)\mathrm{d}x + Q(x,y)\mathrm{d}y = 0 P(x,y)dx+Q(x,y)dy=0

称为齐次的，如果函数 P ~P~ P 和 Q ~Q~ Q 是同次的齐次函数。现在 P(x,y)/Q(x,y) ~P(x,y)/Q(x,y)~ P(x,y)/Q(x,y) 是 0 ~0~ 0 次齐次函数，因此它可写为 φ(yx) ~\varphi(\frac{y}{x})~ φ(xy​) 的形式。于是上述的齐次微分方程可化为形式

dydx=φ(yx)\frac{\mathrm{d}y}{\mathrm{d}x} = \varphi(\frac{y}{x}) dxdy​=φ(xy​)

为了解上述方程，我们引入新的未知数函数

u=yxu = \frac{y}{x} u=xy​

从而

dydx=u+xdudx\frac{\mathrm{d}y}{\mathrm{d}x} = u + x\frac{\mathrm{d}u}{\mathrm{d}x} dxdy​=u+xdxdu​

于是待求方程变为

xdudx=φ(u)−u x\frac{\mathrm{d}u}{\mathrm{d}x} = \varphi(u) - u xdxdu​=φ(u)−u

此时可分离变量，成为

duφ(u)−u=dxx\frac{\mathrm{d}u}{\varphi(u) - u} = \frac{\mathrm{d}x}{x} φ(u)−udu​=xdx​

两边积分，得到

∫duφ(u)−u=ln⁡∣x∣+C\int \frac{\mathrm{d}u}{\varphi(u) - u} = \ln|x| + C ∫φ(u)−udu​=ln∣x∣+C

求出不定积分后再用 y/x ~y/x~ y/x 替换其中的 u ~u~ u ，即得到原方程的通解。注意若 φ(u)−u ~\varphi(u) - u~ φ(u)−u 有一个实零点 u0 ~u_0~ u0​ ，则 y=u0x ~y = u_0 x~ y=u0​x 就是丢失的一个特解，应当补上。

一阶线性方程

如果一阶微分方程中的未知函数 y ~y~ y 及其导函数 y′ ~y^\prime~ y′ 都是一次的，则称方程为一阶线性方程，它可化为标准形式

dydx+P(x)y=Q(x)\frac{\mathrm{d}y}{\mathrm{d}x} + P(x)y = Q(x) dxdy​+P(x)y=Q(x)

若右端的 Q(x) ~Q(x)~ Q(x) 恒为零，则方程称为（一阶）线性齐次方程；否则，方程称为（一阶）线性非齐次方程。注意这里的“齐次”和先前说过的含义完全不同。

线性齐次方程的求解是相当容易的，因为它是可分离变量的：当 y≠0 ~y\neq 0~ y​=0 时，我们有

dyy=−P(x)dx\frac{\mathrm{d}y}{y} = -P(x)\mathrm{d}x ydy​=−P(x)dx

得出

ln⁡∣y∣=−∫P(x)dx+C1\ln |y| = -\int P(x)\mathrm{d}x + C_1 ln∣y∣=−∫P(x)dx+C1​

由此得

y=Ce−∫P(x)dxy = Ce^{-\int P(x)\mathrm{d}x} y=Ce−∫P(x)dx

其中 C=±eC1(≠0) ~C = \pm e^{C_1}(\neq 0)~ C=±eC1​(​=0) 

显然 y=0 ~y = 0~ y=0 是一个丢失的特解，只需令 C=0 ~C = 0~ C=0 就得到了这个解。故通解为

y=Ce−∫P(x)dxy = Ce^{-\int P(x)\mathrm{d}x} y=Ce−∫P(x)dx

其中 C ~C~ C 为任意常数。为求非齐次方程的通解，我们将 C ~C~ C 易变为 x ~x~ x 的函数 C(x) ~C(x)~ C(x) ，即

y=C(x)e−∫P(x)dxy = C(x)e^{-\int P(x)\mathrm{d}x} y=C(x)e−∫P(x)dx

带入原方程得

dC(x)dx=Q(x)e∫P(x)dx\frac{\mathrm{d}C(x)}{\mathrm{d}x} = Q(x)e^{\int P(x)\mathrm{d}x} dxdC(x)​=Q(x)e∫P(x)dx

两边积分得

C(x)=∫Q(x)e∫P(x)dxdx+CC(x) = \int Q(x)e^{\int P(x)\mathrm{d}x}\mathrm{d}x + C C(x)=∫Q(x)e∫P(x)dxdx+C

带入可得非齐次方程的通解为

y=e−∫P(x)dx(∫Q(x)e∫P(x)dxdx+C)y = e^{-\int P(x)\mathrm{d}x}\left(\int Q(x)e^{\int P(x)\mathrm{d}x}\mathrm{d}x + C\right) y=e−∫P(x)dx(∫Q(x)e∫P(x)dxdx+C)

有一些一阶方程，并不是线性方程，但可以通过适当的代换化为线性方程，进而其解易于求得。比如 Bernoulli ~\mathrm{Bernoulli}~ Bernoulli 方程。

dydx+P(x)y=Q(x)yn\frac{\mathrm{d}y}{\mathrm{d}x} + P(x)y = Q(x)y^n dxdy​+P(x)y=Q(x)yn

其中 n ~n~ n 为不等于 0,1 ~0,1~ 0,1 的实数。解方程可采用下面的方法：先以 yn ~y^n~ yn 除方程的两边，得到

y−ndydx+P(x)y1−n=Q(x)y^{-n}\frac{\mathrm{d}y}{\mathrm{d}x} + P(x)y^{1-n} = Q(x) y−ndxdy​+P(x)y1−n=Q(x)

再作代换 u=y1−n ~u = y^{1-n}~ u=y1−n ，则方程化为线性方程

dudx+(1−n)P(x)u=(1−n)Q(x)\frac{\mathrm{d}u}{\mathrm{d}x} + (1-n)P(x)u = (1-n)Q(x) dxdu​+(1−n)P(x)u=(1−n)Q(x)

由此就容易求出方程的通解。

可降阶微分方程

一般的二阶微分方程的形式是

F(x,y,y′,y′′)=0F(x,y,y^\prime,y^{\prime\prime}) = 0 F(x,y,y′,y′′)=0

  • 不显含未知函数

    若方程不显含未知函数 y ~y~ y ，则可写为

    f(x,y′,y′′)=0f(x,y^\prime,y^{\prime\prime}) = 0 f(x,y′,y′′)=0

    令 p=y′ ~p = y^\prime~ p=y′ ，则 y′′=dpdx ~y^{\prime\prime} = \frac{\mathrm{d}p}{\mathrm{d}x}~ y′′=dxdp​ 。此时方程化为一阶方程

    f(x,p,dpdx)=0f\left(x,p,\frac{\mathrm{d}p}{\mathrm{d}x}\right) = 0 f(x,p,dxdp​)=0

    通过求解两次一阶微分方程可解出原方程。

  • 不显含自变量

    若方程不显含自变量 x ~x~ x ，则方程可写为

    g(y,y′,y′′)=0g(y,y^\prime,y^{\prime\prime}) = 0 g(y,y′,y′′)=0

    令 p=y′ ~p = y^\prime~ p=y′ ，则

    y′′=dpdx=dpdydydx=pdpdyy^{\prime\prime} = \frac{\mathrm{d}p}{\mathrm{d}x} = \frac{\mathrm{d}p}{\mathrm{d}y}\frac{\mathrm{d}y}{\mathrm{d}x} = p\frac{\mathrm{d}p}{\mathrm{d}y} y′′=dxdp​=dydp​dxdy​=pdydp​

    于是方程可写为

    g(y,p,pdpdy)=0g\left(y,p,p\frac{\mathrm{d}p}{\mathrm{d}y}\right) = 0 g(y,p,pdydp​)=0

    通过求解两次一阶微分方程可解出原方程。

二阶微分方程

二阶常系数齐次线性微分方程

二阶常系数齐次线性微分方程可以写为一般形式

y′′+py′+qy=0y^{\prime\prime} + py^\prime + qy = 0 y′′+py′+qy=0

其中 p,q ~p,q~ p,q 是实常数。假设方程有形如 y(x)=eλx ~y(x) = e^{\lambda x}~ y(x)=eλx （ λ ~\lambda~ λ 是待定常数）的解，则有

λ2eλx+λpeλx+qeλx=0\lambda^2 e^{\lambda x} + \lambda p e^{\lambda x} + qe^{\lambda x} = 0 λ2eλx+λpeλx+qeλx=0

因而

λ2+pλ+q=0\lambda^2 + p\lambda + q = 0 λ2+pλ+q=0

此方程称为原方程的特征方程。因此，只要 λ ~\lambda~ λ 是特征方程的一个根，就对应二阶常系数线性方程的一个解 y=eλx ~y = e^{\lambda x}~ y=eλx . 根据特征方程求根问题的特性，分以下三种情况讨论：

  • 特征方程有两个不同的实根 λ1,λ2 ~\lambda_1,\lambda_2~ λ1​,λ2​ .

    此时， y1(x)=eλ1x,y2(x)=eλ2x ~y_1(x) = e^{\lambda_1 x},y_2(x) = e^{\lambda_2 x}~ y1​(x)=eλ1​x,y2​(x)=eλ2​x 是原方程的解。此时通解为

    y(x)=c1eλ1x+c2eλ2xy(x) = c_1 e^{\lambda_1 x} + c_2 e^{\lambda_2 x} y(x)=c1​eλ1​x+c2​eλ2​x

  • 特征方程有一对复根 λ1=α+iβ, λ2=α−iβ ~\lambda_1 = \alpha + \mathrm{i}\beta,~\lambda_2 = \alpha - \mathrm{i}\beta~ λ1​=α+iβ, λ2​=α−iβ .

    此时方程 y1(x)=eλ1x,y2(x)=eλ2x ~y_1(x) = e^{\lambda_1 x},y_2(x) = e^{\lambda_2 x}~ y1​(x)=eλ1​x,y2​(x)=eλ2​x 仍然是解，不过是一对复函数解。为了得到实函数的解，由 Euler ~\mathrm{Euler}~ Euler 公式有

    eλ1x=(cos⁡βx+isin⁡βx)eαx,  eλ2x=(cos⁡βx−isin⁡βx)eαxe^{\lambda_1 x} = (\cos\beta x + \mathrm{i}\sin\beta x)e^{\alpha x},~~e^{\lambda_2 x} = (\cos\beta x - \mathrm{i}\sin\beta x)e^{\alpha x} eλ1​x=(cosβx+isinβx)eαx,  eλ2​x=(cosβx−isinβx)eαx

    根据方程解满足线性性可知

    eαxcos⁡βx=eλ1x+eλ2x2,  eαxsin⁡βx=eλ1x−eλ2x2ie^{\alpha x}\cos\beta x = \frac{e^{\lambda_1 x} + e^{\lambda_2 x}}{2},~~e^{\alpha x}\sin\beta x = \frac{e^{\lambda_1 x} - e^{\lambda_2 x}}{2\mathrm{i}} eαxcosβx=2eλ1​x+eλ2​x​,  eαxsinβx=2ieλ1​x−eλ2​x​

    是原方程的两个实函数解而且线性无关，因此构成原方程实函数的基本解组。于是原方程的通解为

    y(x)=(c1cos⁡βx+c2sin⁡βx)eαxy(x) = (c_1 \cos \beta x + c_2\sin \beta x)e^{\alpha x} y(x)=(c1​cosβx+c2​sinβx)eαx

  • 特征方程有一个实重根 λ ~\lambda~ λ .

    此时 y1(x)=eλx ~y_1(x) = e^{\lambda x}~ y1​(x)=eλx 是原方程的解。容易验证，方程 y2(x)=xeλx ~y_2(x) = xe^{\lambda x}~ y2​(x)=xeλx 也是原方程的解，它与 y1 ~y_1~ y1​ 线性无关。于是原方程的通解为

    y(x)=c1eλx+c2xeλxy(x) = c_1 e^{\lambda x} + c_2 x e^{\lambda x} y(x)=c1​eλx+c2​xeλx

二阶常系数非齐次线性微分方程的特解

二阶常系数非齐次线性微分方程可写为如下一般形式

y′′+py′+q=f(x)y^{\prime\prime} + py^\prime + q = f(x) y′′+py′+q=f(x)

  1.  f(x) ~f(x)~ f(x) 的形式为 P(x)eλx ~P(x)e^{\lambda x}~ P(x)eλx 型。 P(x) ~P(x)~ P(x) 是关于 x ~x~ x 的多项式， λ ~\lambda~ λ 可以为零。
    则方程的解有如下形式

    y(x)=xkQ(x)eλxy(x) = x^{k}Q(x)e^{\lambda x} y(x)=xkQ(x)eλx

    其中 Q(x) ~Q(x)~ Q(x) 是和 P(x) ~P(x)~ P(x) 有着同样形式的多项式。例如 P(x)=x2+2x ~P(x) = x^2 + 2x~ P(x)=x2+2x ，则 Q(x)=ax2+bx+c ~Q(x) = ax^2 + bx + c~ Q(x)=ax2+bx+c （ a,b,c ~a,b,c~ a,b,c 为待定系数）。

    • 若 λ ~\lambda~ λ 不是特征根，则 k=0, y(x)=Q(x)eλx ~k = 0,~y(x) = Q(x)e^{\lambda x}~ k=0, y(x)=Q(x)eλx .
    • 若 λ ~\lambda~ λ 是一重根，则 k=1, y=xQ(x)eλx ~k = 1,~y = xQ(x)e^{\lambda x}~ k=1, y=xQ(x)eλx .
    • 若 λ ~\lambda~ λ 是二重根，则 k=2, y=x2Q(x)eλx ~k = 2,~y = x^2 Q(x) e^{\lambda x}~ k=2, y=x2Q(x)eλx .

  2.  f(x) ~f(x)~ f(x) 的形式为 eλxP(x)cos⁡(βx) ~e^{\lambda x}P(x)\cos(\beta x)~ eλxP(x)cos(βx) 或 eλxP(x)sin⁡(βx) ~e^{\lambda x}P(x)\sin(\beta x )~ eλxP(x)sin(βx) 型。

    • 若 α+βi ~\alpha + \beta \mathrm{i}~ α+βi 不是特征根，则解的形式为

      y(x)=eλxQ(x)(Acos⁡βx+Bsin⁡βx)y(x) = e^{\lambda x}Q(x)(A\cos\beta x + B\sin\beta x) y(x)=eλxQ(x)(Acosβx+Bsinβx)

    • 若 α+βi ~\alpha + \beta \mathrm{i}~ α+βi 是特征根，则解的形式为

      y(x)=xeλxQ(x)(Acos⁡βx+Bsin⁡βx)y(x) = xe^{\lambda x}Q(x)(A\cos\beta x + B\sin\beta x) y(x)=xeλxQ(x)(Acosβx+Bsinβx)

将特解的形式带入原方程，可以解出待定常数 A,B ~A,B~ A,B ，由此可得到特解。而二阶常系数非齐次线性微分方程的通解，则可以由对应的齐次方程的通解加上非齐次方程的一个特解得到。
